## 嵌套路由配置

嵌套路由是Vue Router中的重要功能，用于构建具有层级结构的页面布局。

以关于页面为例，创建嵌套路由来展示网站的详细信息。这种结构常用于组织相关但相对独立的页面内容。

### 创建子路由要显示加载的页面

在`src/views/about-sub`目录下分别创建`us.vue`和`info.vue`组件。

#### `us.vue`-关于我们页面

```vue
<template>
    <h3>关于我们</h3>
</template>
```

#### `info.vue`-网站信息页面

```vue
<template>
    <h3>网站信息</h3>
</template>
```

### 在路由配置文件中添加子路由

```javascript
{
    path: '/about',
    name: 'About',
    component: () => import("@/views/About.vue"),
    children: [
        {
            path: 'us',
            name: 'name',
            component: () => import('@/views/about-sub/us.vue')
        },
        {
            path: 'info',
            name: 'info',
            component: () => import('@/views/about-sub/info.vue')
        }
    ]
}
```

### 在关于页面显示子路由

```vue
<template>
    <div class="about">
        <p>这里是关于网站的一些信息</p>
        <RouterLink to="/about/us">关于我们</RouterLink> | 
        <RouterLink to="/about/info">关于信息</RouterLink>
        <RouterView />
    </div>
</template>

<script setup>
// 关于页面逻辑
</script>
```
