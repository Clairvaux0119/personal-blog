## 路由传递参数

在Vue.js单页应用中，页面跳转时常常需要传递数据参数。路由参数允许我们在不同组件之间传递信息，实现数据驱动的页面导航。

以新闻列表为例，点击不同新闻条目时需要跳转到详情页并展示对应的内容：

```html
<ul>
    <li>百度新闻</li>
    <li>网易新闻</li>
    <li>头条新闻</li>
</ul>
```

为了实现点击不同新闻显示对应详情，我们需要在路由跳转时携带标识参数`key`。

### 配置动态路由

#### 第一步：定义带参数的路由规则

```javascript
{
    path: '/news',
    name: 'news',
    component: () => import("@/views/news.vue")
},
{
    // :name是动态参数，可以捕获URL中的对应部分
    // 例如 '/news/百度' 会匹配此路由，且name = "百度"
    path: '/news/:name',
    name: 'news-detail',
    component: () => import("@/views/news-detail.vue")
}
```

#### 第二步：跳转时传递参数

```vue
<template>
    <ul>
        <li><RouterLink to="/news/百度">百度新闻</RouterLink></li>
        <li><RouterLink to="/news/网易">网易新闻</RouterLink></li>
        <li><RouterLink to="/news/头条">头条新闻</RouterLink></li>
    </ul>
</template>
```

>**参数说明**
>
>+ `:name`是参数占位符，`:`表示这是一个动态参数
>+ 可以定义多个参数，如`/news/:category/:id`
>+ 参数会从URL路径中自动提取并传递给组件

#### 第三步：接收和使用参数

在目标组件中，通过路由实例访问传递的参数。

```vue
<!-- 组合式API -->
<template>
    <h3>新闻详情</h3>
    <p>{{ newsName }}</p>
</template>

<script setup>
import { computed } from 'vue';
import { useRoute } from 'vue-router';

// 获取路由实例
const route = useRoute()

// 使用计算属性获取参数
const newsName = computed(() => route.params.name)
</script>
```
